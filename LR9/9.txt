1. BLoC (Business Logic Component) - это паттерн управления состоянием, который используется во фреймворке Flutter для разделения бизнес-логики от представления. Основными компонентами BLoC являются: Stream, Sink, StreamController и BlocProvider.
2. Оповещение о изменении состояния widget’a производится с помощью метода setState(), который перестраивает виджет и вызывает его метод build() для обновления отображения на экране.
3. BLoC инициализируется в основном в методе initState() виджета, который вызывается при создании виджета.
4. Конкретный экземпляр BLoC можно получить, используя провайдер BLoC (BlocProvider), который предоставляет доступ к экземпляру BLoC в дереве виджетов приложения. Для этого нужно создать экземпляр провайдера и указать тип BLoC, который вы хотите получить.
